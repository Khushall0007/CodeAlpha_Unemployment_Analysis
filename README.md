# Unemployment Analysis with Python

## Project Overview

This project analyzes unemployment trends in India using Python. The analysis focuses on unemployment rates, regional patterns, Covid-19 impact, and key insights obtained through data visualization.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Project Workflow

1. Data Cleaning
2. Exploratory Data Analysis
3. Covid-19 Impact Analysis
4. Trend Visualization
5. Insights and Conclusions

## Key Findings

* Unemployment increased significantly during Covid-19.
* Different regions showed different unemployment patterns.
* Visualizations helped identify trends and changes over time.
